- **分类清晰**: 按功能模块和使用场景分类
- **层次分明**: 从概览到详细，从入门到进阶
- **双向链接**: 文档间相互引用，便于导航
- **持续更新**: 随项目发展同步更新

- 使用Markdown格式编写
- 统一的标题层级和格式
- 清晰的代码示例和配置说明
- 丰富的图表和流程图

- 新增功能时同步更新相关文档
- 遵循现有的文档结构和格式
- 提供清晰的示例和说明
- 及时更新过时的信息
